En español: Una interfaz web fundamental que demuestra el dominio de las tecnologías base del desarrollo frontend: HTML5 y CSS3. Este proyecto destaca por la implementación de una estructura semántica clara, el uso de variables modernas en CSS para la gestión de estilos personalizados (:root) y un diseño responsivo mediante el modelo de caja (box-sizing), estableciendo las bases para la creación de aplicaciones web interactivas.

In English: A fundamental web interface that demonstrates mastery of core frontend development technologies: HTML5 and CSS3. This project stands out for its implementation of a clear semantic structure, the use of modern CSS variables for custom style management (:root), and a responsive design through the box model, laying the foundation for creating interactive web applications.
