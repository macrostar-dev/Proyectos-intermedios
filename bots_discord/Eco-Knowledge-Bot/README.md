En español: Un avanzado bot de consulta ecológica para Discord que funciona como una enciclopedia interactiva de reciclaje. El proyecto destaca por su capacidad de procesar grandes volúmenes de datos informativos y responder mediante un sistema híbrido que combina descripciones detalladas sobre sostenibilidad con el envío automatizado de archivos multimedia locales (.jpg). Implementa una lógica de búsqueda optimizada y manejo de rutas de archivos para educar a los usuarios de manera visual y textual.

In English: An advanced ecological query bot for Discord that functions as an interactive recycling encyclopedia. The project stands out for its ability to process large volumes of informative data and respond through a hybrid system that combines detailed sustainability descriptions with the automated delivery of local multimedia files (.jpg). It implements optimized search logic and file path management to educate users both visually and textually.
