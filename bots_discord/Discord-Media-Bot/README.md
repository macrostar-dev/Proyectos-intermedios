En español: Un bot especializado para Discord que integra el manejo de archivos locales y multimedia mediante la librería os de Python. Este proyecto demuestra la capacidad de gestionar directorios, seleccionar archivos de forma aleatoria y enviarlos a través de la API de Discord, permitiendo una interacción dinámica basada en el intercambio de contenido visual (memes) con los usuarios.

In English: A specialized Discord bot that integrates local file and multimedia handling using Python's os library. This project demonstrates the ability to manage directories, randomly select files, and send them through the Discord API, allowing for dynamic interaction based on sharing visual content (memes) with users.
