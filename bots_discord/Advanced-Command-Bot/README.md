En español: Un bot versátil para Discord que utiliza la extensión commands para implementar una interfaz de usuario basada en prefijos. Este proyecto destaca por incluir herramientas de utilidad como cálculos matemáticos, sistemas de azar con dados, toma de decisiones aleatorias y gestión de tareas repetitivas mediante programación asíncrona, ofreciendo una experiencia interactiva y dinámica dentro del servidor.

In English: A versatile Discord bot that utilizes the commands extension to implement a prefix-based user interface. This project stands out for including utility tools such as mathematical calculations, dice-based chance systems, random decision-making, and repetitive task management through asynchronous programming, providing an interactive and dynamic experience within the server.
