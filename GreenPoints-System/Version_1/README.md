En español:
GreenPoints v1.0 - Gestión de Datos en Memoria: Primera fase del sistema de recompensas por reciclaje. Este proyecto se centra en la lógica de programación pura, utilizando diccionarios y estructuras de datos dinámicas en Python para gestionar usuarios, validar credenciales y calcular puntos de forma inmediata. Es una demostración de eficiencia en el manejo de flujos de control y almacenamiento temporal.

In English:
GreenPoints v1.0 - In-Memory Data Management: The initial phase of the recycling rewards system. This project focuses on pure programming logic, using Python dictionaries and dynamic data structures to manage users, validate credentials, and calculate points instantly. It demonstrates efficiency in control flow and temporary data storage.
