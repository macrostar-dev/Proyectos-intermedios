En español:
GreenPoints v2.0 - Persistencia con MySQL: Evolución profesional del sistema hacia una arquitectura cliente-servidor. En esta versión, se integra una base de datos relacional para garantizar que la información de los usuarios y sus puntos sea permanente (CRUD). Incluye validaciones avanzadas, manejo de conexiones SQL y una estructura modular que separa la lógica de negocio de la gestión de datos.

In English:
GreenPoints v2.0 - SQL Persistence: A professional evolution towards a client-server architecture. This version integrates a relational database to ensure user information and points are permanently stored (CRUD). It features advanced validations, SQL connection handling, and a modular structure that separates business logic from data management.
