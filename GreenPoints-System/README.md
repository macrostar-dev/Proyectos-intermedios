SPANISH
GreenPoints es un sistema interactivo diseñado para incentivar el reciclaje mediante un programa de recompensas. 
Este proyecto documenta mi transición técnica desde la gestión de datos en memoria (utilizando diccionarios en Python) hasta la implementación de persistencia profesional mediante bases de datos SQL. 
Es una solución funcional que transforma la lógica de negocio en una herramienta con impacto positivo.

ENGLISH
GreenPoints is an interactive rewards system designed to encourage recycling. 
This project showcases my technical evolution from managing data in memory (using Python dictionaries) to implementing professional data persistence with SQL databases. 
It is a functional solution that transforms business logic into a tool with a positive environmental impact.
